## 🌟 Features

- 📊 Real-time transaction analytics
- 🔍 Advanced search functionality
- 📅 Monthly data filtering
- 📈 Interactive charts (Bar & Pie)
- 🌙 Modern dark theme UI
- 📱 Fully responsive design
- 🔄 Real-time data updates

## 🛠️ Tech Stack

### Frontend
- React.js
- Material-UI
- Chart.js
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
